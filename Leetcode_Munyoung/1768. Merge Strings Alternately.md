# 1768. Merge Strings Alternately

## My solution

```java
class Solution {
    public String mergeAlternately(String word1, String word2) {

        //Declare the length of each word
        int word1Length = word1.length(); 
        int word2Length = word2.length(); 

        //Create Stringbuilder object 
        StringBuilder sb = new StringBuilder();

        //Declare the index start with 0
        int index = 0; 
        
        //Use while loop until index is smaller than the length of each word. 
        while(index<word1Length && index<word2Length){
            sb.append(word1.charAt(index));
            sb.append(word2.charAt(index));
            index++; 
        }
         
        //Add the left part of the word. 
         sb.append(word1.substring(index, word1Length)); 
         sb.append(word2. substring(index, word2Length)); 
         return sb.toString(); 
    }
}
```
