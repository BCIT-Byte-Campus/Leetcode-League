# 238. Product of Array Except Self

## My solution(solution referenced)
```java
class Solution {
    public int[] productExceptSelf(int[] nums) {

        //Declare the value product that multiply all the element in array
        //and Initialize it to 1. 
        int product = 1; 

        //Save the array nums length to N. 
        final int N = nums.length; 

        //Use for loop to multiply all numbers in the array. 
        for(int i =0; i < N; i++){
            product *= nums[i]; 
        }

        //Create answer to save the result. 
        int[] answer = new int[N]; 
        
        for(int i =0; i < N; i++){
            if(nums[i] == 0){
                answer[i] = ifZero(nums, i, N); 
                continue;            
        }
        answer[i] = product/nums[i]; 
    }
    return answer; 
}

//Create ifZero method
public int ifZero(int[] nums, int index, int N){
    int answer = 1; 
    for(int i =0; i < N; i++){
        if(i == index) continue; 
        answer *= nums[i]; 
    }
    return answer; 
}
}
```
