# 1456. Maximum Number of Vowels in a Substring of Given Length

## My solution

```java
class Solution {
    public int maxVowels(String s, int k) {
        //Initialize the maxCount and count 
        int maxCount = 0; 
        int count = 0; 
        
        //Use for loop to iterate over the char of the strirng 's'
        for(int i =0; i < s.length(); i++){
            char c = s.charAt(i); 
            if(isVowel(c)){
                count++;
            }

            //Check starting index is a vowel
            if(i >= k && isVowel(s.charAt(i - k))){
                count--;
            }
            maxCount = Math.max(maxCount, count); 
        }
        return maxCount; 
    }

    //isVowel method to check if it is a vowel 
    private boolean isVowel(char c)
    {
        return c == 'a' || c == 'e' || c == 'i' || c == 'o' || c == 'u';
    }
}
```
