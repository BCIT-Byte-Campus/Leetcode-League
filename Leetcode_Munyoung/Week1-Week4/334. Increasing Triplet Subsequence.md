# 334. Increasing Triplet Subsequence


## My solution
```java
class Solution {
    public boolean increasingTriplet(int[] nums) {

        //1. Return false when the array is smaller than 2. 
        if(nums.length <= 2)
        return false; 

        //Declare the first and second value. 
        //Inintialize it to Max integer value. 
        int first = Integer.MAX_VALUE; 
        int second = Integer.MAX_VALUE; 

        
        for(int number : nums){
            // When the current number is smaller than first
            // Save the number to first. 
            if(number <= first)
            first = number; 

            //When the current number is smaller than second, 
            //Save the number to second. 
            else if(number <= second)
            second = number; 
            
            else
            return true; 
        }
        return false; 
    }
}
```
