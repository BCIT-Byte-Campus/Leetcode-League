# 605. Can Place Flowers

## My solution(solution referenced) 

```java
class Solution {
    public boolean canPlaceFlowers(int[] flowerbed, int n) {
        int i = 0; 
        while (i < flowerbed.length && n != 0) {

            //when the index i is 1, next 1 should be +2. 
            if (flowerbed[i] == 1) {
                i += 2; 
                continue; 
            }

            // 1. i == 0 : check whether the index i is first index 
            // 2. i == flowerbed.length - 1 :check whether the index i is the last index. 
            // 3. n to count the planting flowers. 
            if (((i == 0 && flowerbed[0] == 0) || flowerbed[i - 1] == 0) &&
                ((i == flowerbed.length - 1 && flowerbed[i] == 0) || flowerbed[i + 1] == 0)) {
                n--; 
            }

            // move the next index. 
            i += 1; 
        }
        i++; 
        return n == 0;
    }
}
```
