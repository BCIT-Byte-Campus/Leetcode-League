# 1431. Kids With the Greatest Number of Candies

## My solution

```java
class Solution {
    public List<Boolean> kidsWithCandies(int[] candies, int extraCandies) {

        //Create the ArrayList object which type is boolean
        List<Boolean> list = new ArrayList<>(); 

        //Declare the max candies and initialize it to candies[0]
        int max = candies[0]; 

        //Declare the int n
        int n = candies.length; 

        //First, check the which kid has greater amount of candies
        for(int i =1; i<n; i++){
            if(candies [i] > max){
                    max = candies[i]; 
                }
        }

        //Compare the number of candies. 
        for(int i = 0; i <n; i++){
            list.add(candies[i]+extraCandies >= max); 
        }
        return list; 
    }
}
```
