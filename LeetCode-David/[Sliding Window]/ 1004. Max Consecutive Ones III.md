## [Sliding Window] 1456. Maximum Number of Vowels in a Substring of Given Length

### My solution
Runtime: 128 ms

```py
class Solution(object):
    def maxVowels(self, s, k):
        """
        :type s: str
        :type k: int
        :rtype: int
        """
        vowels = set('aeiou')
        count = sum(1 for char in s[:k] if char in vowels)
        max_count = count

        for i in range(k, len(s)):
            if s[i] in vowels:
                count += 1
            if s[i - k] in vowels:
                count -= 1

            max_count = max(max_count, count)

        return max_count
```

                windowVowels -= 1
