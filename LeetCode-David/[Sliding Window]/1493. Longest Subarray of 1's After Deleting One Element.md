## [Sliding Window] 1493. Longest Subarray of 1's After Deleting One Element

### My solution
Runtime: 340 ms

```py
class Solution(object):
    def longestSubarray(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        prev, curr, max_len = -1, 0, 0
        for num in nums:
            if num == 1:
                curr += 1
            else:
                prev, curr = curr, 0
            max_len = max(max_len, prev + curr)
        return max_len if max_len != len(nums) else max_len - 1
```

        :type nums: List[int]

        """

        res=0
        x=-1
        y=-1
        n=len(nums)
        for i in range(n):
            if nums[i]==0:
                if ind==0:
                    ind=1
                    x=y
                    y=i
                else:
                    res=max(res,i-x-2)
                    x=y
                    y=i
        if ind==0:
            return n-1
        else:
            res=max(res,n-x-2)
            return res
