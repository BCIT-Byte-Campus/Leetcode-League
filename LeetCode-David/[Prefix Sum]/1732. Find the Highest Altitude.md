## [Sliding Window] 1732. Find the Highest Altitude

### My solution
Runtime: 25 ms

```py
class Solution(object):
    def largestAltitude(self, gain):
        """
        :type gain: List[int]
        :rtype: int
        """
        final = [0]
        for i in gain:
            final.append(final[-1] + i)
        return max(final)
```

```py
class Solution(object):
    def largestAltitude(self, gain):
        """
        :type gain: List[int]
        :rtype: int
        """
        allAlts = []
        startAlt = 0
        allAlts.append(0)
        for alt in gain:
            allAlts.append(startAlt + alt)
            startAlt += alt

        return max(allAlts)
```
