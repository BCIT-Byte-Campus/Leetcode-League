## [Queue] 993. Number of Recent Calls

### My solution

```py
class RecentCounter(object):
    def __init__(self):
        self.queue = []

    def ping(self, t):
        while self.queue and self.queue[0] + 3000 < t:
            self.queue.pop(0)
        self.queue.append(t)

        return len(self.queue)
```
