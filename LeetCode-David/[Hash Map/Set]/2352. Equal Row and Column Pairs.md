## [Hash Map / Set] 2352. Equal Row and Column Pairs

### My solution

```py
class Solution(object):
    def equalPairs(self, grid):
        """
        :type grid: List[List[int]]
        :rtype: int
        """
        n = len(grid)
        cnt = Counter()
        for i in range(n):
            t = tuple(grid[i])
            cnt[t] += 1
        
        res = 0
        for i in range(n):
            val = []
            for j in range(n):
                val.append(grid[j][i])
            t = tuple(val)
            res += cnt[t]
        return res
```
