## [Hash Map / Set] 1207. Unique Number of Occurrences

### My solution

```py
class Solution(object):
    def uniqueOccurrences(self, arr):
        """
        :type arr: List[int]
        :rtype: bool
        """
        freq_dict = {}

        for num in arr:
            freq_dict[num] = freq_dict.get(num, 0) + 1

        count_set = set(freq_dict.values())

        return len(count_set) == len(freq_dict)
```
